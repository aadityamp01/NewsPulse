# NewsPulse
## Unleash the current affairs at your fingertips


## Technology Used
 1. Android
 2. Kotlin
 3. Jetpack Compose
 4. MVVM Architecture Components
 5. News API
 6. Coil library
 7. Postman
 8. Retrofit
 9. Dagger-Hilt
 10. Compose navigation


## Requirements
1. Android Version 7.0 and above, Api 24
2. CompileSdkVersion 33
3. MinSdkVersion 24
4. Androidx
